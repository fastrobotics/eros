## Usage
Generate code from template:
```bash
cd auto_template
cookiecutter <template> -o <output_directory> # template: node, class, etc
```

Force generate with no input:
```bash
cd auto_template
cookiecutter <template> -o <output_directory> -f --no-input # template: node, class, etc
```
